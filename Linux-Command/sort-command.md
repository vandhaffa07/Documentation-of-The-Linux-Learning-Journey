# PERINTAH `sort` PADA LINUX

## PENGERTIAN
`sort` adalah perintah dasar di Linux yang digunakan untuk menyusun atau mengurutkan baris-baris teks dalam sebuah file atau output dari perintah lain. Hasil pengurutannya bisa berupa urutan abjad (alfabetis) maupun urutan angka (numerik). Secara default, perintah ini tidak mengubah file asli, melainkan hanya menampilkan hasilnya di layar terminal.

---

## SINTAKS
Sintaks umum perintah `sort` adalah :
```bash
sort [OPTION] [FILE]
```
