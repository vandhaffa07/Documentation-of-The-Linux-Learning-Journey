# PERINTAH `mv` PADA LINUX

## PENGERTIAN
`mv` di Linux adalah perintah serbaguna yang digunakan untuk memindahkan atau mengganti nama file dan direktori.

---

## SINTAKS
Sintak umum perintah `mv` : 
```bash
mv [OPTIONS] [SUMBER] [TUJUAN]
```
---

## OPTION
**1. No option** : Tanpa menggunakan option apapun, perintah
